ValueSet: eHDSI Quantity Unit

Official URL: http://hl7europe.org/fhir/terminology/ehdsi/ValueSet/eHDSIQuantityUnit Version: 0.2.0
Draft as of 2025-02-08 Computable Name: eHDSIQuantityUnit
Other Identifiers: UUID:1.3.6.1.4.1.12559.11.10.1.3.1.42.56

The Value Set is used to describe the discrete countible entity in which a pharmaceutical product or manufactured item is presented.
